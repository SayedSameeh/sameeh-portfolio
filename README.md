# Sameeh.exe

Personal portfolio website built using React, Vite, Tailwind CSS, and Framer Motion.

## Features

- Animated EVE assistant
- Glassmorphism UI
- Interactive projects section
- Skill dashboard
- Timeline
- Contact section
- Responsive design

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
